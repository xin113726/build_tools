## yarn

npm install yarn -g

**安装项目依赖**

```
yarn install
```

**安装依赖包**

```
yarn add [package]
```

**升级依赖包**

```
yarn upgrade [package]
```

**移除依赖包**

```
yarn remove [package]
```
